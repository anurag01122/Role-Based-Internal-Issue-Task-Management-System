The **Role-Based Internal Issue & Task Management System** is a cloud-based web application designed to streamline organizational and team task workflows. Built using the **MERN stack (MongoDB, Express.js, React, and Node.js)**, the platform delivers an intuitive and responsive interface for issue reporting, task assignment, progress tracking, and real-time collaboration. The system caters to administrators and team members through role-specific capabilities that enhance productivity and structured management.

---

## Admin Features

1. **User Management**
   - Create administrator accounts  
   - Add and manage team members  
   - Activate or disable user profiles  

2. **Task & Issue Assignment**
   - Assign tasks to individuals or multiple users  
   - Update task details and progress status  

3. **Task Properties Control**
   - Label tasks as:
     - Todo  
     - In Progress  
     - Completed  
   - Set priority levels:
     - High  
     - Medium  
     - Normal  
     - Low  
   - Create and manage sub-tasks  

4. **Asset Management**
   - Upload task-related assets such as images and documents  

5. **Task Governance**
   - Trash or permanently delete tasks  
   - Restore tasks when required  

---

## User Features

1. **Task Interaction**
   - Update task status  
   - View detailed task and issue information  

2. **Communication**
   - Add comments to task activities  
   - Chat within issue threads  

3. **Dashboard Access**
   - Filter tasks into todo, in progress, or completed  
   - View personally assigned tasks  

---

## General Features

1. **Authentication & Authorization**
   - Secure user login using JWT  
   - Role-based access control  

2. **Profile Management**
   - Update user profile information  
   - Change passwords securely  

3. **Activity Tracking**
   - Real-time task updates  
   - Comment history
